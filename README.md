# RestaurantePRO — Control Gerencial

Sistema de gestión para restaurantes. Funciona en el navegador con Supabase como base de datos.

## Estructura del repositorio

```
├── index.html        ← App principal
├── login.html        ← Pantalla de acceso
├── supabase_db.js    ← Capa de datos (Supabase)
└── README.md
```

## Acceso

URL: `https://TU_USUARIO.github.io/restaurante-pro/login.html`

## Setup inicial

1. Crear cuenta en [supabase.com](https://supabase.com)
2. Ejecutar `supabase_schema.sql` en el SQL Editor
3. Subir estos 4 archivos a GitHub
4. Activar GitHub Pages en Settings → Pages
